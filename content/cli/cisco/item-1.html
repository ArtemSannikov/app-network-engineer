Команды SHOW

Команда show (сокращенно sh) позволяет вывести информацию на экран для дальнейшего её изучения и использования.

show version / sh ver - информация об оборужовании

show ip interfaces brief - информация об портах и vlan

sh interfaces status / sh int status - информация о интерфейсах

sh ip protocols - информация о настроенных протоколах

sh vlan - информация о vlan

sh vlan brief / sh vlan br - информация о vlan

sh users - информация об подключенных пользователях

sh ntp status - отображение сведений о состоянии синхронизации сетевого времени, равноправном узле, с которым синхронизировано устройство, и страте NTP данного устройства

sh ssh - проверки SSH-подключений к устройству

sh ip ssh - информация о ssh

sh mac address-table static - информация о статически прописанных разрешенных адрессах

sh port-security address - на каком порту какой адрес настроен

sh interfaces fastethernet0/1 switchport - статус всех коммутационных портов или указанного порта, включая настройки блокировки портов и защиты портов

sh run | begin <word> - показать все начиная с word

sh run | include <word> - показать все строки с word
